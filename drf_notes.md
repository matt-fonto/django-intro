## Initial steps

1. Setup `requirements.txt` file

```txt
django
djangorestframework
environs
```

2. Setup Django project

```bash
django-admin startproject project_name
cd project_name
```

3. Setup virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

4. Install dependencies

```bash
pip3 install -r requirements.txt
```
